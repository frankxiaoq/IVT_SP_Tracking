This code is a C++ implementation of the tracking algorithm described in "Real-time Robust Object Tracking based on Sparse Representation and Incremental Weighted PCA" by
Fuhao Qiu, Xiangming Xu, and Chunmei Qing (2007).
Author: Fuhao Qiu
Email: frankxiaoq@gmail.com
Date: 2014/2/14

The code need OpenCV library and data files required to run the experiments may be obtained at http://www.votchallenge.net/vot2013/
